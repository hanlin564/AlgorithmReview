# 排序

## [1. 排序数组（912）](https://leetcode-cn.com/problems/sort-an-array/)

### 快速排序

>快速排序的主要思想是通过划分将待排序的序列分成前后两部分，其中前一部分的数据都比后一部分的数据要小，然后再递归调用函数对两部分的序列分别进行快速排序，以此使整个序列达到有序。

```java
class Solution {
    public int[] sortArray(int[] nums) {
        randomizedQuickSort(nums, 0, nums.length - 1);
        return nums;
    }

    private void randomizedQuickSort(int[] nums, int left, int right) {
        if(left < right) {
            //先找随机哨兵并进行划分，然后递归排序哨兵左侧和右侧的部分
            int pos = randomizedPartition(nums, left, right);
            randomizedQuickSort(nums, left, pos - 1);
            randomizedQuickSort(nums, pos + 1, right);
        }
    }

    //这个方法对[left,right]区间进行划分，对下面方法的一个封装，无非是随机化哨兵罢了
    private int randomizedPartition(int[] nums, int left, int right) {
        //i为在[left,right]中随机选择的哨兵的下标，不要忘记加上left
        int i = new Random().nextInt(right - left + 1) + left;
        //把哨兵放在[left,right]区间的最后，也就是和nums[right]交换位置
        swap(nums, right, i);
        //根据哨兵对[left,right]进行划分，比哨兵小的在它左侧，大的在右侧
        return partition(nums, left, right);
    }

    //也是对[left,right]区间进行划分，区间最右侧的元素为哨兵
    //返回划分后哨兵的下标，也即是按升序排序后哨兵应在的位置
    private int partition(int[] nums, int left, int right) {
        //选区间的最后一个元素为哨兵（在该函数调用之前已经把随机找的哨兵放在区间最后了）
        int pivot = nums[right];
        //j是区间[left,right]的比哨兵小的那一部分的右边界
        int i = left - 1;
        for(int j = left; j <= right - 1; j++) {
            //若当前数小于等于哨兵，就让较小部分增大一个空位，把当前数放在多出来的位置上
            if (nums[j] <= pivot) {
                i++;
                swap(nums, i, j);
            }
        }
        //把哨兵放回较小部分和较大部分中间，返回它的下标
        swap(nums, i + 1, right);
        return i + 1;
    }

    //交换nums数组中下标i和j的元素
    private void swap(int[] nums, int i, int j) {
        int tmp = nums[i];
        nums[i] = nums[j];
        nums[j] = tmp;
    }
}
```

复杂度分析

- 时间复杂度：基于随机选取主元的快速排序时间复杂度为期望**O(nlogn)**，其中n为数组的长度。详细证明过程可以见《算法导论》第七章，这里不再大篇幅赘述。

- 空间复杂度：**O(h)**，其中 h 为快速排序递归调用的层数。我们需要额外的O(h) 的递归调用的栈空间，由于划分的结果不同导致了快速排序递归调用的层数也会不同，最坏情况下需**O(n)**的空间，最优情况下每次都平衡，此时整个递归树高度为logn，空间复杂度为**O(logn)**。



### 堆排序

```java
class Solution {
    public int[] sortArray(int[] nums) {
        int len = nums.length;
        //建成大根堆
        heapify(nums);

        //重复下面两个操作，这样可达到一个升序的数组
        for (int i = len - 1; i >= 1; ) {
            //把末位元素和堆顶交换，这样最后一个元素就是最大值
            swap(nums, 0, i);
            i--;
            //重新调整0~i-1的序列为一个大根堆
            siftDown(nums, 0, i);
        }
        return nums;
    }

    //将数组转化为有序的堆
    private void heapify(int[] nums) {
        int len = nums.length;
        //(len - 1) / 2是从堆的最后一个节点往前数的第一个非叶子节点的数组下标
        //遍历这些非叶子结点，必要时使它们下沉
        for (int i = (len - 1) / 2; i >= 0; i--) {
            siftDown(nums, i, len - 1);
        }
    }

    //k为要下沉的节点下标，end为区间的后边界
    //该方法会比较节点k与它的子节点的值，若存在比它大的，就交换两者的位置
    //交换会一直进行直到没有子节点比k大了为止(满足大根堆定义)
    private void siftDown(int[] nums, int k, int end) {
        //先说下这个2k+1是啥
        //通过画图可知，一个堆中节点的数组下标为k，那么它的左子节点的数组下标为2k+1
        while (2 * k + 1 <= end) {
            //j为左子节点的下标
            int j = 2 * k + 1;
            //右子节点没越界 且 右子节点大于左子节点
            //j++，即指向更大的右子节点
            if (j + 1 <= end && nums[j + 1] > nums[j]) {
                j++;
            }
            //如果子节点大于节点k，需要交换两个节点的位置，即把节点值大的放上去
            //放上去的值可能是左子节点也可能是右子节点
            if (nums[j] > nums[k]) {
                swap(nums, j, k);
            } else {//若子节点不比父节点大，就退出循环
                break;
            }
            //更新当前节点下标为j，即当前节点下标置为为比父节点更大的字节点下标
            //把被置换下去的节点与更下方的字节点做比较
            k = j;
        }
    }

    //交换nums数组中下标i和j的元素
    private void swap(int[] nums, int i, int j) {
        int tmp = nums[i];
        nums[i] = nums[j];
        nums[j] = tmp;
    }
}
```

复杂度分析

- 时间复杂度：**O(nlogn)**。初始化建堆的时间复杂度为O(n)，建完堆以后需要进行n−1 次调整，一次调整（即 maxHeapify） 的时间复杂度为O(logn)，那么n−1次调整即需要O(nlogn) 的时间复杂度。因此，总时间复杂度为 O(n+nlogn)=O(nlogn)。

- 空间复杂度：**O(1)**。只需要常数的空间存放若干变量。



### 归并排序

```java
class Solution {
    //用于临时存放归并排序结果的数组，里面只有一部分存放了正确排序的元素
    //准确来说是[0,count]区间内的元素有效
    //count为当前两个数组归并后的总长度
    int[] tmp;

    public int[] sortArray(int[] nums) {
        tmp = new int[nums.length];
        mergeSort(nums, 0, nums.length - 1);
        return nums;
    }

    //对[left,right]区间进行归并排序
    //随着递归深入，nums会被切分成只有1个元素的小数组
    //对这些小数组进行两两合并，然后再把它们合并的结果进行两两合并
    //最终得到有序的数组
    private void mergeSort(int[] nums, int left, int right) {
        if (left >= right) {
            return;
        }
        //用位运算求中间位置的下标
        int mid = left + ((right - left) >> 1);
        //对左右两边进行归并排序
        mergeSort(nums, left, mid);
        mergeSort(nums, mid + 1, right);
        //i指向[left,mid]中的元素，j指向[mid+1,right]中的元素
        int i = left, j = mid + 1;
        //count用于控制往tmp中添加元素的顺序
        int count = 0;
        while (i <= mid && j <= right) {
            //哪个指针指向的元素比较小，就往tmp添加它
            //注意要让指针右移
            if (nums[i] <= nums[j]) {
                tmp[count++] = nums[i++];
            } else {
                tmp[count++] = nums[j++];
            }
        }
        //当一个指针走到末尾时上面的循环跳出
        //这时就把另外一个区间的元素全部按顺序添加到tmp中就行了
        while (i <= mid) {
            tmp[count++] = nums[i++];
        }
        while (j <= right) {
            tmp[count++] = nums[j++];
        }
        //把tmp中有效的部分按顺序添加进nums
        for (int k = 0; k < right - left + 1; k++) {
            nums[k + left] = tmp[k];
        }
    }
}
```

复杂度分析

- 时间复杂度：**O(nlogn)**
- 空间复杂度：**O(n)**
